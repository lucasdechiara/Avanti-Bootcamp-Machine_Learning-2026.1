## BOOTCAMP - MACHINE LEARNING - 2026.1
**Ementa:** O curso propõe conceitos fundamentais e aplicações práticas para a capacitação do aluno na criação e avaliação de modelos preditivos capazes de classificar ou segmentar informações na área de machine learning. O curso introduz conceitos sobre operações lógicas, estrutura de dados, análise e visualização de dados e o desenvolvimento de arquiteturas de redes neurais convolucionais para a criação de modelos preditivos.


**Carga Horária:** 80 h 

**Objetivos Gerais:** Qualificar o(a) aluno(a) na compreensão dos aspectos fundamentais e principais algoritmos de aprendizado de máquina, por meio do desenvolvimento de técnicas que auxiliam na resolução de problemas na área de machine learning.

**Objetivos Específicos:** Apresentar a linguagem python para machine learning, apresentar formas de visualização e análise de dados e apresentar técnicas de classificação e segmentação de imagens.


**Conteudo Programatico**
* Lógica de Programação: Fundamentos de lógica de programação.
* Noções de Python: Sintaxe de Python, definição de variáveis, funções, listas, dicionários.
* Matrizes: Manipulação de matrizes, operações básicas.
* Aritmética Básica: Soma, subtração, multiplicação, divisão, frações e porcentagens.
